# 🎓 Attendance Calculator (75% Rule)

A simple and responsive Attendance Calculator that helps students track their attendance percentage and determine how many classes they need to attend to maintain the minimum 75% attendance requirement. The application provides quick calculations and an easy-to-use interface for students. The project is built using HTML, CSS, and JavaScript. :contentReference[oaicite:0]{index=0}

## 🚀 How It Works

1. Enter the **Total Classes Held**.
2. Enter the **Classes Attended**.
3. Enter the **Classes Per Day**.
4. Click on **Calculate**.
5. View your attendance status and required classes.
